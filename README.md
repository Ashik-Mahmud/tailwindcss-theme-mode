# Toggle Theme changer using tailwind css

## Installation

```bash
npm install
```

## Usage

```bash

npm run dev

```

## Code Snippet

```bash
<ToggleTheme rounded={false} style="text" size="md"  />

- rounded={false} // for rounded or not
- style="text" // for text | icon | both | shmooz-ai
- size="md" // for sm | md | lg | xl

```
